O Projeto

Porque escolher colaborar em um game?

Muitos desenvolvedores que estão iniciando suas carreiras possuem uma enorme vontade de colaborar em projetos open-soure, contudo se veem incapazes de realizar tal feito ou não encontram um projeto com uma proposta que possa agradá-los. Pensando em sanar tais problemas transformei meu trabalho de faculdade em um jogo open-source de desenvolvimento colaborativo.

Desenvolvido em Ruby, linguagem bastante agradável para iniciantes e com uma curva de aprendizagem baixa, o projeto também chama atenção por ser um game, fato que torna o desenvolvimento algo ainda mais prazeroso. É incrível imaginar que é possível falar para os seus amigos que você colaborou no desenvolvimento de um game e poder usá-los para experimentar o jogo.

Os pré-requisitos para colaborar no projeto é conhecer o paradigma de Orientação a Objetos - que é utilizado pela biblioteca Gosu de desenvolvimento de jogos, porém se você ainda não souber esse assunto não desanime. O repositório possui vários links de estudo possibilitando, assim, um ganho de conhecimento ainda maior para aqueles que se disponibilizam em contribuir.

Fique a vontade para acessar o código, analisá-lo, ler os comentários, ver as issues do projeto e criar algumas. Este repositório é nosso! Que tal começar a colaborar agora mesmo?


Eu gostaria de colaborar no projeto!
